# Kafe/Məkan Admin Paneli üçün Lazımi İnputlar

Admin panelində məkanın idarə olunması üçün lazım olan bütün sahələrin (inputs) tam siyahısı.

## 1. Əsas Məlumatlar (Basic Info)
- **Məkanın Adı:** (Text) - Məsələn: "Nexus Prime Esports"
- **Qısa Təsvir (Slogan):** (Text) - Listinqdə görünən qısa məlumat.
- **Ətraflı Təsvir (About):** (Textarea) - Məkan haqqında geniş məlumat.
- **Kateqoriya:** (Dropdown/Select) - Məsələn: Lounge, Gaming, Kafe, Restoran.
- **Premium Statusu:** (Switch/Toggle) - Məkanın "Premium" olub-olmaması.

## 2. Ünvan və Kordinat (Location)
- **Tam Ünvan:** (Text) - Məsələn: "Bakı, Nizami küç. 45"
- **Şəhər/Rayon:** (Dropdown)
- Kordinat

## 3. Media və Şəkillər
- **Əsas Şəkil (Hero Image):** (Image Upload/URL) - Əsas qapaq şəkli.
- **Qalereya:** (Multi-Image Upload/URLs) - Məkanın daxili şəkilləri.

## 4. İş Saatları (Opening Hours)
- **Hər gün üçün ayrıca:** (Time Picker)
  - Bazar ertəsi: Başlanğıc - Sonu
  - Çərşənbə axşamı: Başlanğıc - Sonu ... və s.
- **7/24 Rejimi:** (Checkbox)

## 5. Qiymət və Xidmətlər (Pricing & Services)
- **Başlanğıc Qiymət:** (Number) - Məsələn: 10
- **Xidmətlər (Amenities):** (Multi-select/Checkbox list)
  - 📶 Pulsuz WiFi
  - 🎮 PS5 / Gaming PC
  - ☕ Qəhvə və Çay
  - 🍔 Qida/Snack
  - ❄️ Kondisioner
  - 🚗 Parkinq
  - 🚪 Şəxsi Otaqlar (VIP)
  Customda elave ede biler
  xidmetlerin iconunu ozu elave ede biler img 

## 6. Əlaqə Məlumatları (Contact)
- **Telefon Nömrəsi:** (Phone Input)
- **E-poçt:** (Email Input)
- **Instagram Linki:** (URL)
- **WhatsApp Nömrəsi:** (Phone Input)
- **Veb-sayt:** (URL)
